# Heart Disease Prediction

This project uses Machine Learning to predict the likelihood of heart disease in patients based on various medical attributes. The model is trained using Logistic Regression and provides predictions through a simple user interface.

## Features
- Data preprocessing and cleaning
- Feature scaling using StandardScaler
- Heart disease prediction using Logistic Regression
- User-friendly GUI using Tkinter
- Model evaluation using classification metrics

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Tkinter
- Jupyter Notebook

## Dataset
Heart Disease Dataset

## Model
Logistic Regression
